# Expense Tracker 💰

## 📌 Project Overview
Expense Tracker is a Java-based application that helps users track and manage their daily expenses efficiently.

## 🚀 Features
- Add new expenses
- View expense list
- Calculate total expenses
- Simple and user-friendly interface

## 🛠️ Tech Stack
- Java
- OOP Concepts

## ▶️ How to Run
1. Open project in Eclipse
2. Run Main.java
3. Follow on-screen instructions
